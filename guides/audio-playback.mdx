Sound makes gameplay feel alive — a crunch when a bullet lands, a musical sting when you clear a level, ambient effects that set the mood. Fraxel plays audio through the Web Audio API via the `<audio-player>` node. The workflow mirrors textures: load a sound file to get a symbol ID, then reference that symbol in your JSX.

## Loading a Sound

```tsx
import { loadSound } from 'fraxel/assets'

const SHOOT_SFX = await loadSound('/assets/shoot.mp3')
const JUMP_SFX = await loadSound('/assets/jump.ogg')
const BG_MUSIC = await loadSound('/assets/music.mp3')
```

`loadSound` fetches the file, decodes it with `AudioContext.decodeAudioData`, and returns a `symbol`. Identical URLs are deduplicated — calling `loadSound` twice with the same URL returns the same symbol without a second network request.

## The `<audio-player>` Node

Place `<audio-player>` anywhere in your scene tree and pass the sound's symbol ID. Call `play()` on the node to start playback:

```tsx Gun.tsx
import { useAudio, useClickable, useTrigger } from 'fraxel/hooks'
import { loadSound } from 'fraxel/assets'

const SHOOT_SFX = await loadSound('/assets/shoot.mp3')

function Gun() {
  const audio = useAudio()
  const btn = useClickable()

  useTrigger(btn.clicked, () => {
    audio.play()
  })

  return (
    <sprite textureId={GUN_TEX}>
      <audio-player ref={audio} soundId={SHOOT_SFX} volume={0.8} />
      <clickable ref={btn} size={[48, 32]} />
    </sprite>
  )
}
```

## `<audio-player>` Props

<ParamField path="soundId" type="symbol" required>
  The symbol returned by `loadSound()`. Required — the node won't play without it.
</ParamField>

<ParamField path="loop" type="boolean" default="false">
  Whether the sound loops continuously. Set to `true` for background music.
</ParamField>

<ParamField path="volume" type="number" default="1">
  Playback volume from `0` (silent) to `1` (full). Values above `1` are technically allowed but can distort.
</ParamField>

<ParamField path="playbackRate" type="number" default="1">
  Playback speed. `2` = double speed (higher pitch), `0.5` = half speed (lower pitch). Useful for varied sound effects.
</ParamField>

<ParamField path="persistUntilEnd" type="boolean" default="false">
  When `true`, the node defers its own destruction until the sound finishes playing. Use this for one-shot sound effects attached to nodes that get destroyed on impact.
</ParamField>

## Methods and Getters

```tsx
const audio = useAudio()

// Start playback (optional offset in seconds)
audio.play()
audio.play(1.5) // start 1.5 seconds in

// Pause without resetting position
audio.pause()

// Stop and reset to beginning
audio.stop()

// Read current state
console.log(audio.playing())    // boolean (reactive)
```

## Events

| Event | Callback | Description |
|-------|----------|-------------|
| `ended` | `() => void` | Fires when a non-looping sound reaches the end |
| `error` | `(err: Error) => void` | Fires if playback fails (e.g. unsupported format) |

```tsx
useTrigger(audio.ended, () => {
  console.log('Sound finished playing')
})
```

## Browser Autoplay Policy

Browsers block audio until the user has interacted with the page. If you try to play sound before a click or keypress, it will be silently blocked. Resume the `AudioContext` after the first user gesture:

```tsx
import { getAudioContext } from 'fraxel/audio'
import { useClickable, useTrigger } from 'fraxel/hooks'

function GameScene() {
  const clickable = useClickable()

  useTrigger(clickable.clicked, () => {
    const ctx = getAudioContext()
    if (ctx.state === 'suspended') {
      ctx.resume()
    }
  })

  return (
    <sprite ref={clickable} textureId={START_BTN}>
      <clickable size={[120, 40]} />
    </sprite>
  )
}
```

<Warning>
  Always call `getAudioContext().resume()` inside a user interaction handler (click, keydown, etc.) before attempting to play audio. Add this to your start button or first-click handler.
</Warning>

## The `useAudio` Derived Hook

`useAudio` wraps `AudioPlayer` with reactive state and control methods, so you can build play/pause toggles and reactive UI without manual event wiring:

```tsx MusicPlayer.tsx
import { useAudio, useClickable, useTrigger, useComputed } from 'fraxel/hooks'

const BG_MUSIC = await loadSound('/assets/music.mp3')

function MusicPlayer() {
  const audio = useAudio()
  const btn = useClickable()

  const label = useComputed(() => (audio.playing() ? 'Pause' : 'Play'))

  useTrigger(btn.clicked, () => {
    audio.playing() ? audio.pause() : audio.play()
  })

  return (
    <transform>
      <audio-player ref={audio} soundId={BG_MUSIC} loop volume={0.4} />
      <sprite textureId={MUSIC_BTN}>
        <clickable ref={btn} size={[64, 24]} />
        <text position={[8, 16]} text={label} style={{ fontSize: 12, foregroundColor: '#fff' }} />
      </sprite>
    </transform>
  )
}
```

| Property | Type | Description |
|----------|------|-------------|
| `ref` | `NodeReference` | Pass to `<audio-player ref={ref}>` |
| `playing` | `SignalGetter<boolean>` | `true` while audio is playing |
| `play()` | `() => void` | Start playback |
| `pause()` | `() => void` | Pause without resetting |
| `stop()` | `() => void` | Stop and reset |

## Complete Example

```tsx GameAudio.tsx
import { useAudio, useMount } from 'fraxel/hooks'
import { loadSound } from 'fraxel/assets'
import { getAudioContext } from 'fraxel/audio'

const BG_MUSIC = await loadSound('/assets/music.mp3')
const SHOOT_SFX = await loadSound('/assets/shoot.mp3')
const HIT_SFX = await loadSound('/assets/hit.mp3')

function GameAudio() {
  const music = useAudio()
  const sfx = useAudio()
  const hitSfx = useAudio()

  useMount(() => {
    const ctx = getAudioContext()
    if (ctx.state === 'suspended') ctx.resume()
    music.play()
  })

  // Expose sfx nodes so other components can call .play()
  // (typically via FraxelScript or trigger events)

  return (
    <transform>
      <audio-player ref={music} soundId={BG_MUSIC} loop volume={0.25} />
      <audio-player ref={sfx} soundId={SHOOT_SFX} volume={0.7} />
      <audio-player ref={hitSfx} soundId={HIT_SFX} volume={0.9} />
    </transform>
  )
}
```
