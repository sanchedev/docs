title: '<raycast> — Directional ray collider detection node'
sidebarTitle: '<raycast>'
description: "Project a ray from the node's world position to detect the first matching collider. Fires colliderEntered and colliderExited as the hit changes."
---

Use `<raycast>` to detect colliders along a straight line from the node's world position. Define a `direction` vector (in pixels) and a list of groups to check against. The ray queries collision groups directly — not the spatial hash — so it sees every registered collider in the target groups regardless of distance. Subscribe to `colliderEntered` and `colliderExited` to react when the first hit changes, or poll `getCollider()` each frame.

```tsx
import { useRayCast, useTrigger } from 'fraxel'

function LineOfSight() {
  const ray = useRayCast()

  useTrigger(ray.colliderEntered, (col) => {
    console.log('Enemy spotted at', col.globalPosition)
  })

  useTrigger(ray.colliderExited, () => {
    console.log('Lost sight')
  })

  return (
    <raycast
      ref={ray}
      direction={[200, 0]} // 200 px to the right
      collidesWith={['enemy']}
    />
  )
}
```

## Props

<ParamField path="direction" type="Reactive<VectorLike>" required>
  The endpoint of the ray **relative to the node's world position**. The ray
  travels from `globalPosition` to `globalPosition + direction`. Measured in
  pixels, so `[200, 0]` casts 200 px to the right.

Accepts a reactive `SignalGetter<VectorLike>` for a dynamically rotating or
scaling ray.

```tsx
<raycast direction={[150, 0]} collidesWith={['enemy']} />
<raycast direction={[0, 80]} collidesWith={['ground']} />  {/* downward */}
<raycast direction={() => aimVector()} collidesWith={['target']} />
```

</ParamField>

<ParamField path="collidesWith" type="string[]" required>
  Groups to test against. The ray only reports hits from colliders whose `group`
  intersects this list. At least one entry is required.

```tsx
<raycast direction={[100, 0]} collidesWith={['enemy', 'obstacle']} />
```

</ParamField>

<ParamField path='position' type='Reactive<VectorLike>'>
  Local offset of the ray origin relative to the parent node. Defaults to `[0,
  0]`. Set this to offset the origin (e.g. to the gun barrel tip).
</ParamField>

<ParamField path='gameMode' type='GameMode'>
  Controls how this node behaves when the game is paused. Defaults to
  `GameMode.INHERIT`. Use `GameMode.ALWAYS` for nodes that should keep updating
  while paused.
</ParamField>

<ParamField path='ref' type='NodeReference<PrimaryNode.RayCast>'>
  Attach a node reference to call `getCollider()` or subscribe to events.
</ParamField>

## Computed properties

| Property         | Type                   | Description                                               |
| ---------------- | ---------------------- | --------------------------------------------------------- |
| `direction`      | `Vector2`              | The resolved direction vector (mutable if not reactive).  |
| `collidesWith`   | `string[]` (read-only) | The group filter list as passed at construction.          |
| `length`         | `number` (read-only)   | The Euclidean length of the `direction` vector in pixels. |
| `globalPosition` | `Vector2`              | The world-space origin of the ray.                        |

## Methods

| Method        | Signature                | Description                                                                                                                          |
| ------------- | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------ |
| `getCollider` | `() => Collider \| null` | Returns the collider currently intersected by the ray, or `null` if none. Can be polled each frame instead of subscribing to events. |

## Events

| Event name        | Callback signature             | Description                                                                                                                       |
| ----------------- | ------------------------------ | --------------------------------------------------------------------------------------------------------------------------------- |
| `colliderEntered` | `(collider: Collider) => void` | Fires when a new collider is first hit by the ray. The same event fires on the hit collider as well.                              |
| `colliderExited`  | `(collider: Collider) => void` | Fires when the previously detected collider is no longer in the ray's path. The same event fires on the exiting collider as well. |
| `started`         | `() => void`                   | Fires once after `start()`.                                                                                                       |
| `updated`         | `(delta: number) => void`      | Fires every frame.                                                                                                                |
| `drawed`          | `(delta: number) => void`      | Fires every frame when the node is being drawn.                                                                                   |
| `destroyed`       | `() => void`                   | Fires on removal.                                                                                                                 |
| `zIndexChanged`   | `(zIndex: number) => void`     | Fires when the node's `zIndex` value changes.                                                                                     |

<CodeGroup>

```tsx Ground detection for a platformer
import { useRigidBody, useRayCast, useTrigger, useSignal } from 'fraxel'
import { shapes } from 'fraxel'

function Player() {
  const body = useRigidBody()
  const groundRay = useRayCast()
  const [isGrounded, setGrounded] = useSignal(false)

  useTrigger(groundRay.colliderEntered, () => setGrounded(true))
  useTrigger(groundRay.colliderExited, () => setGrounded(false))

  return (
    <body ref={body} mass={1} position={[100, 50]}>
      <sprite textureId={PLAYER_TEX} />
      <collider
        shape={shapes.rectangle(14, 24)}
        group={['player']}
        collidesWith={['ground']}
      />
      {/* Short downward ray from the feet */}
      <raycast
        ref={groundRay}
        position={[7, 24]}
        direction={[0, 4]}
        collidesWith={['ground']}
      />
    </body>
  )
}
```

```tsx Rotating laser sight
import { useRayCast, useTrigger, useSignal } from 'fraxel'

function Turret() {
  const ray = useRayCast()
  const [angle, setAngle] = useSignal(0)

  useTrigger(ray.updated, (delta) => {
    setAngle((a) => a + delta * 90) // rotate 90°/s
  })

  return (
    <transform position={[200, 200]}>
      <sprite textureId={TURRET_TEX} />
      <raycast
        ref={ray}
        direction={() => [
          Math.cos((angle() * Math.PI) / 180) * 300,
          Math.sin((angle() * Math.PI) / 180) * 300,
        ]}
        collidesWith={['enemy']}
      />
    </transform>
  )
}
```

</CodeGroup>

<Note>
  Ray casts query **all registered colliders** in the target groups, not just
  nearby ones. There is no distance cap other than the length of the `direction`
  vector. Keep `direction` short if you want a finite-range sensor.
</Note>

<Tip>
  Enable `gameConfig.testOptions.showRayCasts = true` to draw rays as red arrows
  in the canvas during development.
</Tip>
