Core hooks are the fundamental building blocks of every Fraxel component. They wire reactive state, node references, event subscriptions, and game controls into your JSX components. All hooks must be called at the top level of a component function — never inside conditionals or loops.

---

## useSignal

Creates a reactive signal with an initial value. When the signal's value changes, any `useEffect` or `useComputed` that read from it will automatically re-run.

```ts
useSignal<T>(initialValue: T): [SignalGetter<T>, SignalSetter<T>]
```

<ParamField path="initialValue" type="T" required>
  The initial value of the signal.
</ParamField>

**Returns** a tuple `[getter, setter]`:

| Item | Type | Description |
|---|---|---|
| `getter` | `() => T` | Call to read the current value. Reading inside `useEffect` or `useComputed` establishes a reactive dependency. |
| `setter` | `(value: T) => void` | Call to update the value and notify subscribers. |

```tsx
import { useSignal, useEffect } from 'fraxel/hooks'

function HealthBar() {
  const [health, setHealth] = useSignal(100)

  useEffect(() => {
    console.log('Health is now:', health())
  })

  return (
    <transform>
      <geometry fillColor={[1 - health() / 100, health() / 100, 0, 1]} />
    </transform>
  )
}
```

<Note>
Subscribers are automatically cleaned up when the node is destroyed. You do not need to manually unsubscribe.
</Note>

## useComputed

Creates a derived signal that recomputes automatically whenever any signal it reads changes.

```ts
useComputed<T>(fn: () => T): SignalGetter<T>
```

<ParamField path="fn" type="() => T" required>
  A function that derives a value from one or more signal getters. Re-runs whenever a dependency changes.
</ParamField>

**Returns** `SignalGetter<T>` — a read-only getter for the derived value.

```tsx
import { useSignal, useComputed } from 'fraxel/hooks'

function CooldownBar() {
  const [elapsed, setElapsed] = useSignal(0)

  // Automatically updates when elapsed changes
  const progress = useComputed(() => elapsed() / 3)

  return (
    <geometry
      shape={shapes.rectangle(progress() * 100, 10)}
      fillColor={[1, 1, 0, 1]}
    />
  )
}
```

<Tip>
Use `useComputed` to transform or combine signals without managing intermediate state. It is more efficient than deriving values inside `useEffect`.
</Tip>

## useEffect

Runs a side-effect function when the node starts and again whenever any signal it reads changes. Multiple synchronous signal changes are batched — the effect runs once per microtask, not once per change.

```ts
useEffect(fn: () => void | (() => void)): void
```

<ParamField path="fn" type="() => void | (() => void)" required>
  The effect function. If it returns a function, that cleanup function runs before the next execution and when the node is destroyed.
</ParamField>

```tsx
import { useSignal, useEffect } from 'fraxel/hooks'

function Tracker() {
  const [x, setX] = useSignal(0)
  const [y, setY] = useSignal(0)

  useEffect(() => {
    console.log(`Position: ${x()}, ${y()}`)

    // Optional cleanup
    return () => {
      console.log('Effect cleaned up')
    }
  })

  return <transform />
}
```

**Batching behaviour:** If `x`, `y`, and `z` all change in the same synchronous block, the effect runs exactly once before the next frame:

```tsx
setX(1)
setY(2)
setZ(3)
// Effect runs once, not three times
```

Re-executions are deferred via `queueMicrotask` and always run before the next animation frame.

## useNode

Creates a typed reference to a specific node type. Pass the returned reference to a node's `ref` prop to capture the live node instance.

```ts
useNode<T extends PrimaryNode>(type: T): NodeReference<T>
```

<ParamField path="type" type="PrimaryNode" required>
  The node type constant from the `PrimaryNode` enum, e.g. `PrimaryNode.Sprite`, `PrimaryNode.RigidBody`.
</ParamField>

**Returns** `NodeReference<T>` — a typed wrapper around the node that becomes populated once the node mounts.

| Property | Type | Description |
|---|---|---|
| `ref.node` | `NodeInstances[T]` | The live node instance. Throws `NodeNotInitializedError` if accessed before mount. |
| `ref.signal` | `SignalGetter<NodeInstances[T] \| null>` | Reactive getter — `null` before mount, the node instance after. |

```tsx
import { useNode, useEvent } from 'fraxel/hooks'
import { PrimaryNode } from 'fraxel'

function Enemy() {
  const collider = useNode(PrimaryNode.Collider)

  useEvent(collider, 'colliderEntered', (other) => {
    console.log('Hit by:', other)
  })

  return (
    <collider
      ref={collider}
      shape={shapes.circle(8)}
      group={['enemy']}
      collidesWith={['projectile']}
    />
  )
}
```

## useEvent

Subscribes to a named event on a node reference. The subscription is automatically removed when the node is destroyed.

```ts
useEvent<N extends PrimaryNode, K extends keyof Events<N>>(
  node: NodeReference<N>,
  eventName: K,
  listener: Events<N>[K]
): void
```

<ParamField path="node" type="NodeReference<N>" required>
  The node reference whose event you want to subscribe to.
</ParamField>
<ParamField path="eventName" type="string" required>
  The name of the event. TypeScript will infer valid event names from the node type.
</ParamField>
<ParamField path="listener" type="function" required>
  The callback to invoke when the event fires. Argument types are inferred from the node's event definitions.
</ParamField>

```tsx
import { useNode, useEvent } from 'fraxel/hooks'
import { Input, PrimaryNode } from 'fraxel'

function Player() {
  const body = useNode(PrimaryNode.RigidBody)

  useEvent(body, 'updated', (delta) => {
    // Runs every physics frame
    body.node.velocity.x = Input.getKeyAxis('ArrowRight', 'ArrowLeft') * 120
  })

  return <rigid-body ref={body} />
}
```

## useMount

Runs a function exactly once when the node starts. If the function returns a cleanup function, that cleanup runs when the node is destroyed. Unlike `useEffect`, `useMount` does not re-run when signals change.

```ts
useMount(fn: () => void | (() => void)): void
```

<ParamField path="fn" type="() => void | (() => void)" required>
  Function to run on mount. Return a cleanup function to run on destroy.
</ParamField>

```tsx
import { useMount } from 'fraxel/hooks'

function BackgroundMusic() {
  const audio = new AudioContext()

  useMount(() => {
    console.log('Scene loaded — starting music')

    return () => {
      console.log('Scene destroyed — stopping music')
      audio.close()
    }
  })

  return <transform />
}
```

## useSpawn

Returns a function that dynamically appends JSX nodes as children of a given container node at runtime. The spawn context (providers, etc.) is captured at call time.

```ts
useSpawn<T extends PrimaryNode>(node: NodeReference<T>): (jsx: Fraxel.Node) => void
```

<ParamField path="node" type="NodeReference<T>" required>
  Reference to the container node that spawned children will be added to.
</ParamField>

**Returns** `(jsx: Fraxel.Node) => void` — call with any JSX element to append it as a child.

```tsx
import { useNode, useSpawn } from 'fraxel/hooks'
import { PrimaryNode } from 'fraxel'

function Spawner() {
  const container = useNode(PrimaryNode.Transform)
  const spawn = useSpawn(container)

  return (
    <transform ref={container}>
      <clickable
        size={[64, 32]}
        onClick={() => spawn(<Enemy position={[Math.random() * 200, 0]} />)}
      />
    </transform>
  )
}
```

## useGame

Returns a `GameControls` object that lets you play, pause, destroy, or change the active scene from inside any component.

```ts
useGame(): GameControls
```

**Returns** `GameControls`:

| Method | Signature | Description |
|---|---|---|
| `play` | `() => void` | Resumes the game loop. |
| `pause` | `() => void` | Pauses the game loop. |
| `destroy` | `() => void` | Stops the game and releases all resources. |
| `changeScene` | `(name: string) => Promise<void>` | Destroys the current scene and loads the named one. |
| `preloadScene` | `(name: string) => Promise<() => void>` | Preloads a scene in the background; returns a setter function. |
| `getSize` | `() => Vector2` | Returns the game canvas dimensions as a `Vector2`. |

```tsx
import { useGame } from 'fraxel/hooks'

function PauseButton() {
  const game = useGame()

  return (
    <clickable
      size={[48, 16]}
      onClick={() => game.pause()}
    />
  )
}
```

## useChild

Resolves a child node reference by traversing a path of node IDs from the root node of the current component.

```ts
useChild<T extends PrimaryNode>(
  path: (string | symbol)[],
  type: T
): NodeReference<T>
```

<ParamField path="path" type="(string | symbol)[]" required>
  Array of node IDs forming the path from the component root to the target child.
</ParamField>
<ParamField path="type" type="PrimaryNode" required>
  The expected node type at the end of the path.
</ParamField>

**Returns** `NodeReference<T>` — populated after the node starts.

```tsx
import { useChild, useMount } from 'fraxel/hooks'
import { PrimaryNode } from 'fraxel'

function CharacterRig() {
  const weapon = useChild(['body', 'arm', 'weapon'], PrimaryNode.Sprite)

  useMount(() => {
    weapon.node.visible = true
  })

  return (
    <transform id="body">
      <transform id="arm">
        <sprite id="weapon" textureId={SWORD} />
      </transform>
    </transform>
  )
}
```

## useScript

Retrieves the `FraxelScript` instance attached to a node reference as a reactive signal. The signal value updates reactively as the node mounts.

```ts
useScript<T extends FraxelScript<PrimaryNode>>(
  node: NodeReference<PrimaryNode>
): SignalGetter<T | undefined>
```

<ParamField path="node" type="NodeReference<PrimaryNode>" required>
  The node reference whose script you want to access.
</ParamField>

**Returns** `SignalGetter<T | undefined>` — `undefined` before the node mounts, the script instance after.

```tsx
import { useNode, useScript, useEffect } from 'fraxel/hooks'
import { PrimaryNode } from 'fraxel'
import { PlayerScript } from './player-script'

function HUD() {
  const player = useNode(PrimaryNode.Sprite)
  const script = useScript<PlayerScript>(player)

  useEffect(() => {
    const s = script()
    if (s) console.log('Player health:', s.health)
  })

  return (
    <sprite ref={player} textureId={PLAYER} script={new PlayerScript()} />
  )
}
```

## useAction

Provides reactive frame-by-frame state for a single input action. All three signals update at the start of every frame, so they can be read inside `useEffect`.

```ts
useAction(action: symbol): {
  pressed: SignalGetter<boolean>
  justPressed: SignalGetter<boolean>
  justUnpressed: SignalGetter<boolean>
}
```

<ParamField path="action" type="symbol" required>
  An action symbol created with `Input.createAction()`.
</ParamField>

| Property | Type | Description |
|---|---|---|
| `pressed` | `SignalGetter<boolean>` | `true` while the action key is held down. |
| `justPressed` | `SignalGetter<boolean>` | `true` only on the first frame the key is pressed. |
| `justUnpressed` | `SignalGetter<boolean>` | `true` only on the first frame the key is released. |

```tsx
import { Input, PrimaryNode } from 'fraxel'
import { useAction, useEffect, useNode } from 'fraxel/hooks'

const Jump = Input.createAction({ key: ' ' })

function Player() {
  const body = useNode(PrimaryNode.RigidBody)
  const { justPressed } = useAction(Jump)

  useEffect(() => {
    if (justPressed()) {
      body.node.applyImpulse([0, -400])
    }
  })

  return <rigid-body ref={body} />
}
```

## useActionAxis

Returns a reactive axis value derived from two opposing input actions. Updates every frame.

```ts
useActionAxis(negative: symbol, positive: symbol): SignalGetter<number>
```

<ParamField path="negative" type="symbol" required>
  Action that maps to `-1` (e.g. move left).
</ParamField>
<ParamField path="positive" type="symbol" required>
  Action that maps to `+1` (e.g. move right).
</ParamField>

**Returns** `SignalGetter<number>` — returns `-1`, `0`, or `1`.

```tsx
import { Input } from 'fraxel'
import { useActionAxis, useNode, useEvent } from 'fraxel/hooks'
import { PrimaryNode } from 'fraxel'

const MoveLeft  = Input.createAction({ key: 'a' })
const MoveRight = Input.createAction({ key: 'd' })

function Player() {
  const body = useNode(PrimaryNode.RigidBody)
  const axis = useActionAxis(MoveLeft, MoveRight)

  useEvent(body, 'updated', () => {
    body.node.velocity.x = axis() * 180
  })

  return <rigid-body ref={body} />
}
```

## useTrigger / createTrigger

`createTrigger` creates a typed pub/sub channel. `useTrigger` subscribes to that channel inside a component with automatic cleanup on destroy.

```ts
createTrigger<T extends any[]>(): Trigger<T>
useTrigger<T extends any[]>(trigger: Trigger<T>, fn: (...args: T) => void): void
```

<ParamField path="trigger" type="Trigger<T>" required>
  The trigger created by `createTrigger`.
</ParamField>
<ParamField path="fn" type="(...args: T) => void" required>
  Callback that fires when `trigger.emit(...)` is called.
</ParamField>

**Trigger class methods:**

| Method | Description |
|---|---|
| `trigger.connect(fn)` | Manually add a listener. |
| `trigger.disconnect(fn)` | Manually remove a listener. |
| `trigger.emit(...args)` | Call all connected listeners with the given arguments. |

```tsx
import { createTrigger, useTrigger, useCollider, useEffect } from 'fraxel/hooks'
import { shapes } from 'fraxel'

// Declare at module level so it is shared between components
const onEnemyKilled = createTrigger<[enemyId: number]>()

// Parent listens
function ScoreManager() {
  useTrigger(onEnemyKilled, (id) => {
    console.log('Enemy killed:', id)
  })
  return <transform />
}

// Child emits
function Enemy({ id }: { id: number }) {
  const { ref, colliding } = useCollider()

  useEffect(() => {
    if (colliding()) {
      onEnemyKilled.emit(id)
    }
  })

  return <collider ref={ref} shape={shapes.circle(8)} group={['enemy']} collidesWith={['bullet']} />
}
```

## createContext / useContext

Creates a typed context object with a `Provider` component. Descendant components can read the value with `useContext`.

```ts
createContext<T>(defaultValue: T): ContextCreated<T>
useContext<T>(ctx: ContextCreated<T>): T
```

<ParamField path="defaultValue" type="T" required>
  The value returned by `useContext` when no `Provider` is found in the ancestor tree.
</ParamField>

**`ContextCreated<T>` shape:**

| Property | Description |
|---|---|
| `ctx.Provider` | A component that accepts `value` and `children` props. Wrap subtrees that should receive the context value. |

```tsx
import { createContext, useContext } from 'fraxel/hooks'

const DifficultyCtx = createContext<'easy' | 'hard'>('easy')

function GameScene() {
  return (
    <DifficultyCtx.Provider value="hard">
      <Enemy />
    </DifficultyCtx.Provider>
  )
}

function Enemy() {
  const difficulty = useContext(DifficultyCtx)
  const speed = difficulty === 'hard' ? 200 : 100

  return <transform />
}
```

## useRef

Creates a mutable reference object that persists across renders but does not trigger reactivity when changed. Analogous to a plain mutable variable scoped to the component.

```ts
useRef<T>(value: T): Reference<T>
```

<ParamField path="value" type="T" required>
  The initial value stored in the reference.
</ParamField>

**Returns** `Reference<T>` — an object with a single mutable `current` property.

```tsx
import { useRef, useCollider, useEffect } from 'fraxel/hooks'
import { shapes } from 'fraxel'

function Projectile() {
  const bounces = useRef(0)

  const { ref, colliding } = useCollider()

  useEffect(() => {
    if (colliding()) {
      bounces.current++
      if (bounces.current >= 3) {
        // destroy self
      }
    }
  })

  return (
    <collider
      ref={ref}
      shape={shapes.circle(4)}
      group={['projectile']}
      collidesWith={['wall']}
    />
  )
}
```
