Context hooks provide a way to share state across components without prop drilling.

---

## createContext

Creates a typed context object with a `Provider` component. Descendant components can read the value with `useContext`.

```ts
createContext<T>(defaultValue: T): ContextCreated<T>
```

<ParamField path="defaultValue" type="T" required>
  The value returned by `useContext` when no `Provider` is found in the ancestor tree.
</ParamField>

**`ContextCreated<T>` shape:**

| Property | Description |
|---|---|
| `ctx.Provider` | A component that accepts `value` and `children` props. Wrap subtrees that should receive the context value. |

```tsx
import { createContext, useContext } from 'fraxel/hooks'

const DifficultyCtx = createContext<'easy' | 'hard'>('easy')

function GameScene() {
  return (
    <DifficultyCtx.Provider value="hard">
      <Enemy />
    </DifficultyCtx.Provider>
  )
}

function Enemy() {
  const difficulty = useContext(DifficultyCtx)
  const speed = difficulty === 'hard' ? 200 : 100

  return <transform />
}
```

---

## useContext

Retrieves the value of a context. Returns the default value if no `Provider` is found in the ancestor tree.

```ts
useContext<T>(contextCreated: ContextCreated<T>): T
```

<ParamField path="contextCreated" type="ContextCreated<T>" required>
  The context created with `createContext`.
</ParamField>

**Returns** `T` — the current context value.

```tsx
import { createContext, useContext } from 'fraxel/hooks'

const ThemeCtx = createContext<'light' | 'dark'>('light')

function Child() {
  const theme = useContext(ThemeCtx)
  return <text text={theme} />
}
```

<Note>
Context values are static — they do not trigger reactivity when changed. If you need reactive shared state, use signals at module scope or the Trigger system.
</Note>
