Derived hooks are composed from core hooks to provide domain-specific, higher-level abstractions. They do not introduce unique engine functionality — they simplify common patterns that would otherwise require several lines of `useNode` + `useEvent` + `useSignal`. Each derived hook that targets a specific node type accepts an optional `NodeReference` parameter; if you omit it, the hook creates a fresh reference internally.

---

## useCondition

Creates a reactive boolean that flips to `true` when one event fires and back to `false` when another fires. Internally composes `useSignal` + two `useEvent` calls.

```ts
useCondition<N extends PrimaryNode>(
  node: NodeReference<N>,
  agreeEvent: keyof Events<N>,
  disagreeEvent: keyof Events<N>,
  defaultValue?: boolean
): SignalGetter<boolean>
```

<ParamField path="node" type="NodeReference<N>" required>
  The node whose events toggle the condition.
</ParamField>
<ParamField path="agreeEvent" type="string" required>
  The event name that sets the condition to `true`.
</ParamField>
<ParamField path="disagreeEvent" type="string" required>
  The event name that sets the condition to `false`.
</ParamField>
<ParamField path="defaultValue" type="boolean">
  Initial value before any event fires. Defaults to `false`.
</ParamField>

**Returns** `SignalGetter<boolean>`.

<CodeGroup>
```tsx Hover detection
import { useNode, useCondition, useComputed } from 'fraxel/hooks'
import { PrimaryNode } from 'fraxel'

function Button() {
  const clickable = useNode(PrimaryNode.Clickable)
  const hovered = useCondition(clickable, 'mouseEntered', 'mouseExited')
  const brightness = useComputed(() => (hovered() ? 1.2 : 1))

  return (
    <sprite textureId={BTN} brightness={brightness}>
      <clickable ref={clickable} size={[64, 32]} />
    </sprite>
  )
}
```
```tsx Ray cast detection
import { useNode, useCondition } from 'fraxel/hooks'
import { PrimaryNode } from 'fraxel'

function EnemyScanner() {
  const ray = useNode(PrimaryNode.RayCast)
  const detected = useCondition(ray, 'colliderEntered', 'colliderExited')

  useEffect(() => {
    if (detected()) console.log('Enemy in range!')
  })

  return <ray-cast ref={ray} direction={[120, 0]} collidesWith={['enemy']} />
}
```
</CodeGroup>

## useMatch

Creates a computed signal that maps the current value of a signal to a corresponding entry in a record — like a reactive `switch` expression.

```ts
useMatch<T extends number | string | symbol, K>(
  signal: SignalGetterLike<T>,
  record: Record<T, K>,
  defaultValue?: K
): SignalGetter<K | undefined>
```

<ParamField path="signal" type="SignalGetterLike<T>" required>
  A signal getter (or raw value) providing the key to look up.
</ParamField>
<ParamField path="record" type="Record<T, K>" required>
  A plain object mapping each possible signal value to a result.
</ParamField>
<ParamField path="defaultValue" type="K">
  Returned when the signal's value is not found in the record.
</ParamField>

**Returns** `SignalGetter<K | undefined>`.

```tsx
import { useSignal, useMatch } from 'fraxel/hooks'

function Character() {
  const [state, setState] = useSignal<'idle' | 'walking' | 'jumping'>('idle')

  const animName = useMatch(state, {
    idle:    'idle-anim',
    walking: 'walk-anim',
    jumping: 'jump-anim',
  })

  return (
    <animation-player
      animations={() => anims}
      currentAnim={animName}
    />
  )
}
```

## useWhen

Creates a computed signal that switches between two values based on a boolean signal — a reactive ternary expression.

```ts
useWhen<T>(
  signal: SignalGetterLike<boolean>,
  whenTrue: Reactive<T>,
  whenFalse: Reactive<T>
): SignalGetter<T>
```

<ParamField path="signal" type="SignalGetterLike<boolean>" required>
  A signal getter providing the boolean condition.
</ParamField>
<ParamField path="whenTrue" type="Reactive<T>" required>
  Value returned when the signal is `true`. Can be a plain value or another signal getter.
</ParamField>
<ParamField path="whenFalse" type="Reactive<T>" required>
  Value returned when the signal is `false`.
</ParamField>

**Returns** `SignalGetter<T>`.

```tsx
import { useCondition, useWhen } from 'fraxel/hooks'

function InteractiveSprite() {
  const clickable = useNode(PrimaryNode.Clickable)
  const hovered = useCondition(clickable, 'mouseEntered', 'mouseExited')

  const brightness = useWhen(hovered, 1.3, 1.0)
  const cursor     = useWhen(hovered, 'pointer', 'default')

  return (
    <sprite textureId={ICON} brightness={brightness}>
      <clickable ref={clickable} size={[32, 32]} />
    </sprite>
  )
}
```

## useClickable

A node-specific derived hook for the `Clickable` node. Provides the node reference, a reactive `hovered` boolean, and the current pointer `position` over the element.

```ts
useClickable(ref?: NodeReference<PrimaryNode.Clickable>): {
  ref:      NodeReference<PrimaryNode.Clickable>
  hovered:  SignalGetter<boolean>
  position: SignalGetter<Vector2>
}
```

<ParamField path="ref" type="NodeReference<PrimaryNode.Clickable>">
  An optional existing node reference to reuse. If omitted, a new one is created.
</ParamField>

**Return value:**

| Property | Type | Description |
|---|---|---|
| `ref` | `NodeReference<PrimaryNode.Clickable>` | Pass to the `<clickable ref={...}>` prop. |
| `hovered` | `SignalGetter<boolean>` | `true` while the pointer is over the clickable area. |
| `position` | `SignalGetter<Vector2>` | Current pointer position over the element in game coordinates. |

```tsx
import { useClickable, useComputed } from 'fraxel/hooks'

function Button() {
  const { ref, hovered } = useClickable()
  const brightness = useComputed(() => (hovered() ? 1.15 : 1))

  return (
    <sprite textureId={BTN} brightness={brightness}>
      <clickable ref={ref} size={[80, 24]} onClick={handleClick} />
    </sprite>
  )
}
```

## useTimer

A node-specific derived hook for the `Timer` node. Provides the node reference, reactive time and progress signals, and `play`, `pause`, `stop` controls.

```ts
useTimer(ref?: NodeReference<PrimaryNode.Timer>): {
  ref:      NodeReference<PrimaryNode.Timer>
  time:     SignalGetter<number>
  progress: SignalGetter<number>
  play:     Timer['play']
  pause:    Timer['pause']
  stop:     Timer['stop']
}
```

<ParamField path="ref" type="NodeReference<PrimaryNode.Timer>">
  Optional existing node reference to reuse.
</ParamField>

**Return value:**

| Property | Type | Description |
|---|---|---|
| `ref` | `NodeReference<PrimaryNode.Timer>` | Pass to `<timer ref={...}>`. |
| `time` | `SignalGetter<number>` | Current elapsed time in seconds. |
| `progress` | `SignalGetter<number>` | `0`–`1`, computed as `time / duration`. |
| `play()` | `() => void` | Start or resume the timer. |
| `pause()` | `() => void` | Pause the timer without resetting. |
| `stop()` | `() => void` | Stop and reset the timer to `0`. |

```tsx
import { useTimer } from 'fraxel/hooks'
import { shapes } from 'fraxel'

function CooldownIndicator() {
  const { ref, progress, play } = useTimer()

  useMount(() => play())

  return (
    <transform>
      <timer ref={ref} duration={5} />
      <geometry
        shape={shapes.rectangle(progress() * 80, 8)}
        fillColor={[1 - progress(), progress(), 0, 1]}
      />
    </transform>
  )
}
```

## useRayCast

A node-specific derived hook for the `RayCast` node. Provides the node reference, a reactive `detected` boolean, and the currently-detected collider signal.

```ts
useRayCast(ref?: NodeReference<PrimaryNode.RayCast>): {
  ref:      NodeReference<PrimaryNode.RayCast>
  detected: SignalGetter<boolean>
  collider: SignalGetter<Collider | null>
}
```

<ParamField path="ref" type="NodeReference<PrimaryNode.RayCast>">
  Optional existing node reference to reuse.
</ParamField>

**Return value:**

| Property | Type | Description |
|---|---|---|
| `ref` | `NodeReference<PrimaryNode.RayCast>` | Pass to `<ray-cast ref={...}>`. |
| `detected` | `SignalGetter<boolean>` | `true` while any collider is intersecting the ray. |
| `collider` | `SignalGetter<Collider \| null>` | The first collider currently detected, or `null`. |

```tsx
import { useRayCast, useComputed } from 'fraxel/hooks'

function Peashooter() {
  const { ref, detected, collider } = useRayCast()

  const targetName = useComputed(() => collider()?.node.name ?? 'none')

  useEffect(() => {
    if (detected()) {
      console.log('Target acquired:', targetName())
    }
  })

  return (
    <ray-cast
      ref={ref}
      direction={[200, 0]}
      collidesWith={['zombie']}
    />
  )
}
```

## useCollider

A node-specific derived hook for the `Collider` node. Tracks whether the collider is currently overlapping any other collider and exposes the full set of detected colliders.

```ts
useCollider(ref?: NodeReference<PrimaryNode.Collider>): {
  ref:               NodeReference<PrimaryNode.Collider>
  colliding:         SignalGetter<boolean>
  detectedColliders: SignalGetter<Set<Collider>>
}
```

<ParamField path="ref" type="NodeReference<PrimaryNode.Collider>">
  Optional existing node reference to reuse.
</ParamField>

**Return value:**

| Property | Type | Description |
|---|---|---|
| `ref` | `NodeReference<PrimaryNode.Collider>` | Pass to `<collider ref={...}>`. |
| `colliding` | `SignalGetter<boolean>` | `true` while one or more colliders are overlapping. |
| `detectedColliders` | `SignalGetter<Set<Collider>>` | The live set of all overlapping colliders. |

```tsx
import { useCollider, useComputed } from 'fraxel/hooks'
import { shapes } from 'fraxel'

function DamageZone() {
  const { ref, colliding, detectedColliders } = useCollider()

  useEffect(() => {
    if (colliding()) {
      for (const other of detectedColliders()) {
        console.log('Damaging:', other.node.name)
      }
    }
  })

  return (
    <collider
      ref={ref}
      shape={shapes.rectangle(48, 48)}
      group={['zone']}
      collidesWith={['player']}
    />
  )
}
```

## useAnimation

A node-specific derived hook for the `AnimationPlayer` node. Provides reactive animation state and control methods.

```ts
useAnimation(ref?: NodeReference<PrimaryNode.AnimationPlayer>): {
  ref:        NodeReference<PrimaryNode.AnimationPlayer>
  animName:   SignalGetter<string | null>
  frameIndex: SignalGetter<number>
  ended:      SignalGetter<boolean>
  play:       AnimationPlayer['play']
  stop:       AnimationPlayer['stop']
  setNext:    AnimationPlayer['setNext']
}
```

<ParamField path="ref" type="NodeReference<PrimaryNode.AnimationPlayer>">
  Optional existing node reference to reuse.
</ParamField>

**Return value:**

| Property | Type | Description |
|---|---|---|
| `ref` | `NodeReference<PrimaryNode.AnimationPlayer>` | Pass to `<animation-player ref={...}>`. |
| `animName` | `SignalGetter<string \| null>` | Name of the currently-playing animation, or `null` when stopped. |
| `frameIndex` | `SignalGetter<number>` | Index of the current frame. |
| `ended` | `SignalGetter<boolean>` | `true` from the moment the animation finishes until a new one starts. |
| `play(name?)` | `(name?: string) => void` | Start playing an animation by name, or restart the current one. |
| `stop()` | `() => void` | Stop playback. |
| `setNext(name?)` | `(name?: string) => void` | Queue an animation to play after the current one ends. |

```tsx
import { useAnimation, useActionAxis } from 'fraxel/hooks'
import { Input } from 'fraxel'

const Left  = Input.createAction({ key: 'a' })
const Right = Input.createAction({ key: 'd' })

function Character() {
  const { ref, animName, play } = useAnimation()
  const axis = useActionAxis(Left, Right)

  useEffect(() => {
    const next = axis() !== 0 ? 'walk' : 'idle'
    if (animName() !== next) play(next)
  })

  return (
    <animation-player
      ref={ref}
      animations={() => ({ idle: idleFrames, walk: walkFrames })}
      currentAnim="idle"
    />
  )
}
```

## useAudio

A node-specific derived hook for the `AudioPlayer` node. Provides a reactive `playing` boolean and `play`, `pause`, `stop` controls.

```ts
useAudio(ref?: NodeReference<PrimaryNode.AudioPlayer>): {
  ref:     NodeReference<PrimaryNode.AudioPlayer>
  playing: SignalGetter<boolean>
  play:    AudioPlayer['play']
  pause:   AudioPlayer['pause']
  stop:    AudioPlayer['stop']
}
```

<ParamField path="ref" type="NodeReference<PrimaryNode.AudioPlayer>">
  Optional existing node reference to reuse.
</ParamField>

**Return value:**

| Property | Type | Description |
|---|---|---|
| `ref` | `NodeReference<PrimaryNode.AudioPlayer>` | Pass to `<audio-player ref={...}>`. |
| `playing` | `SignalGetter<boolean>` | `true` while audio is playing (between `started` and `ended` events). |
| `play()` | `() => void` | Start or resume playback. |
| `pause()` | `() => void` | Pause without seeking to the beginning. |
| `stop()` | `() => void` | Stop and reset to the beginning. |

```tsx
import { useAudio, useClickable } from 'fraxel/hooks'

function MusicToggle() {
  const { ref: audio, playing, play, pause } = useAudio()
  const { ref: btn, hovered } = useClickable()

  return (
    <sprite ref={btn} textureId={playing() ? PAUSE_ICON : PLAY_ICON}>
      <clickable size={[32, 32]} onClick={() => (playing() ? pause() : play())} />
      <audio-player ref={audio} soundId={MUSIC} loop />
    </sprite>
  )
}
```

## useRigidBody

A node-specific derived hook for the `RigidBody` node. Provides reactive velocity and grounded state, plus imperative force and velocity methods.

```ts
useRigidBody(ref?: NodeReference<PrimaryNode.RigidBody>): {
  ref:          NodeReference<PrimaryNode.RigidBody>
  velocity:     SignalGetter<Vector2>
  isGrounded:   SignalGetter<boolean>
  applyForce:   (force: VectorLike) => void
  applyImpulse: (impulse: VectorLike) => void
  setVelocity:  (v: VectorLike) => void
}
```

<ParamField path="ref" type="NodeReference<PrimaryNode.RigidBody>">
  Optional existing node reference to reuse.
</ParamField>

**Return value:**

| Property | Type | Description |
|---|---|---|
| `ref` | `NodeReference<PrimaryNode.RigidBody>` | Pass to `<rigid-body ref={...}>`. |
| `velocity` | `SignalGetter<Vector2>` | Current velocity in pixels/second as a reactive `Vector2`. |
| `isGrounded` | `SignalGetter<boolean>` | `true` when the body is resting on a surface. |
| `applyForce(force)` | `(VectorLike) => void` | Add a continuous force (px/s²) accumulated this frame. |
| `applyImpulse(impulse)` | `(VectorLike) => void` | Add an instant velocity change (px/s). |
| `setVelocity(v)` | `(VectorLike) => void` | Override the current velocity. |

```tsx
import { useRigidBody } from 'fraxel/hooks'
import { Input, shapes } from 'fraxel'

const Jump  = Input.createAction({ key: ' ' })
const Left  = Input.createAction({ key: 'a' })
const Right = Input.createAction({ key: 'd' })

function Player() {
  const { ref, isGrounded, applyImpulse, setVelocity } = useRigidBody()
  const { justPressed } = useAction(Jump)
  const axis = useActionAxis(Left, Right)

  useEffect(() => {
    setVelocity([axis() * 160, ref.node.velocity.y])
    if (justPressed() && isGrounded()) {
      applyImpulse([0, -420])
    }
  })

  return (
    <rigid-body ref={ref}>
      <sprite textureId={PLAYER} />
      <collider shape={shapes.rectangle(14, 20)} group={['player']} collidesWith={['ground']} />
    </rigid-body>
  )
}
```

## usePartialNode

A low-level utility used internally by all node-specific derived hooks. Returns the provided `NodeReference` as-is, or creates a new one via `useNode` if none is given.

```ts
usePartialNode<T extends PrimaryNode>(
  type: T,
  node?: NodeReference<T>
): NodeReference<T>
```

You typically do not call this directly — it is the pattern that makes optional `ref` parameters work in hooks like `useClickable`, `useTimer`, etc.

```tsx
// useClickable internals — roughly:
function useClickable(clickable?: NodeReference<PrimaryNode.Clickable>) {
  const ref = usePartialNode(PrimaryNode.Clickable, clickable)
  const hovered = useCondition(ref, 'mouseEntered', 'mouseExited')
  // ...
}
```
