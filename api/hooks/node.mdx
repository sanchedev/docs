Node hooks create typed references to JSX nodes. Pass the returned reference to a node's `ref` prop. Fraxel populates it when the node is mounted.

Use `reference.node` for direct access when needed. Prefer the reactive properties, methods, and triggers exposed by the reference when they exist.

---

## Base `NodeReference<T>`

All node hooks return a reference that extends `NodeReference<T>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `node` | `T` | Live node instance. Throws `NodeNotInitializedError` before mount. |
| `signal` | `SignalGetter<T \| null>` | Reactive getter for the live node. Returns `null` before mount and after destroy. |
| `started` | `Trigger<[]>` | Fires when the node starts. |
| `updated` | `Trigger<[delta: number]>` | Fires every update. |
| `drawed` | `Trigger<[delta: number]>` | Fires every draw. |
| `destroyed` | `Trigger<[]>` | Fires when the node is destroyed. |
| `script(scriptClass, noTrack?)` | `InstanceType<K> \| undefined` | Returns the attached `FraxelScript` if it matches the class. |
| `spawn(jsx)` | `(jsx: Fraxel.Node) => void` | Renders JSX and appends the resulting nodes as children. |

```tsx
const group = useGroup()

useTrigger(group.started, () => {
  group.spawn(<Enemy />)
})

return <group ref={group} />
```

## `Node2DReference<T>`

Hooks for 2D nodes extend `Node2DReference<T>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `position` | `SignalGetter<Vector2>` | Current local position. |
| `setPosition` | `(value: VectorLike) => void` | Sets local position. |
| `rotation` | `SignalGetter<number>` | Current local rotation. |
| `setRotation` | `(value: number) => void` | Sets local rotation. |

This base applies to `useTransform`, `useSprite`, `useText`, `useGeometry`, `useCollider`, `useRigidBody`, `useCamera`, `useClickable`, and `useRayCast`.

---

## `useTransform()`

```ts
useTransform(): TransformReference
```

`TransformReference` extends `Node2DReference<PrimaryNode.Transform>` and adds no extra members.

```tsx
import { useTransform, useTrigger } from 'fraxel'

function Mover() {
  const transform = useTransform()

  useTrigger(transform.updated, () => {
    transform.setPosition([transform.position().x + 1, transform.position().y])
  })

  return <transform ref={transform} />
}
```

## `useSprite()`

```ts
useSprite(): SpriteReference
```

`SpriteReference` extends `Node2DReference<PrimaryNode.Sprite>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `source` | `SignalGetter<Region>` | Current source region. |
| `setSource` | `(value: Region) => void` | Sets the source region. |
| `flipX`, `flipY` | `SignalGetter<boolean>` | Flip state. |
| `setFlipX`, `setFlipY` | `(value: boolean) => void` | Sets flip state. |
| `brightness`, `grayscale`, `contrast`, `saturate`, `hueRotate`, `invert`, `opacity` | `SignalGetter<number>` | Filter values. |
| `setBrightness`, `setGrayscale`, `setContrast`, `setSaturate`, `setHueRotate`, `setInvert`, `setOpacity` | `(value: number) => void` | Sets filter values. |
| `modulate` | `SignalGetter<Color>` | Current modulate tint. |
| `setModulate` | `(value: ColorLike) => void` | Sets modulate tint. |

```tsx
import PLAYER from '../assets/player.png?texture'
import { useSprite, useTrigger } from 'fraxel'

function PlayerSprite() {
  const sprite = useSprite()

  useTrigger(sprite.started, () => {
    sprite.setOpacity(0.8)
  })

  return <sprite ref={sprite} textureId={PLAYER} />
}
```

## `useText()`

```ts
useText(): TextReference
```

`TextReference` extends `Node2DReference<PrimaryNode.Text>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `text` | `SignalGetter<string>` | Current text value. |
| `setText` | `(text: string) => void` | Sets the text value. |

```tsx
import { useText, useEffect } from 'fraxel'

function ScoreLabel({ score }: { score: () => number }) {
  const label = useText()

  useEffect(() => {
    label.setText(`Score: ${score()}`)
  })

  return <text ref={label} text="Score: 0" />
}
```

## `useGeometry()`

```ts
useGeometry(): GeometryReference
```

`GeometryReference` extends `Node2DReference<PrimaryNode.Geometry>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `shape` | `SignalGetter<Shape>` | Current rendered shape. |
| `fillColor` | `SignalGetter<Color>` | Current fill color. |
| `strokeColor` | `SignalGetter<Color \| undefined>` | Current stroke color. |
| `strokeWidth` | `SignalGetter<number>` | Current stroke width. |
| `size` | `SignalGetter<Vector2>` | Computed shape size. |

`GeometryReference` exposes reactive reads only. Change geometry through reactive props or `geometry.node` when you need direct control.

```tsx
import { shapes, useGeometry, useEffect } from 'fraxel'

function DebugBox() {
  const geometry = useGeometry()

  useEffect(() => {
    console.log('Geometry size:', geometry.size())
  })

  return <geometry ref={geometry} shape={shapes.rectangle(32, 32)} />
}
```

## `useCollider()`

```ts
useCollider(): ColliderReference
```

`ColliderReference` extends `Node2DReference<PrimaryNode.Collider>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `colliding` | `SignalGetter<boolean>` | `true` when overlapping at least one collider. |
| `detectedColliders` | `SignalGetter<Set<Collider>>` | Current colliders being touched. |
| `colliderEntered` | `Trigger<[other: Collider]>` | Fires when a collision starts. |
| `colliderExited` | `Trigger<[other: Collider]>` | Fires when a collision ends. |

```tsx
import { shapes, useCollider, useTrigger } from 'fraxel'

function Coin() {
  const collider = useCollider()

  useTrigger(collider.colliderEntered, () => {
    collider.node.destroy()
  })

  return <collider ref={collider} shape={shapes.circle(8)} group={['coin']} collidesWith={['player']} />
}
```

## `useRigidBody()`

```ts
useRigidBody(): RigidBodyReference
```

`RigidBodyReference` extends `Node2DReference<PrimaryNode.RigidBody>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `velocity` | `SignalGetter<Vector2>` | Current body velocity. |
| `mass` | `SignalGetter<number>` | Current mass. |
| `friction` | `SignalGetter<number>` | Current friction coefficient. |
| `bounce` | `SignalGetter<number>` | Current bounce value. |
| `isStatic` | `SignalGetter<boolean>` | Whether the body is static. |
| `useGravity` | `SignalGetter<boolean>` | Whether gravity affects the body. |
| `applyForce(force)` | `(force: VectorLike) => void` | Applies continuous force. |
| `applyImpulse(impulse)` | `(impulse: VectorLike) => void` | Applies instant velocity change. |
| `setVelocity(value)` | `(value: VectorLike) => void` | Sets velocity directly. |

```tsx
import { Input, shapes, useAction, useRigidBody, useUpdate } from 'fraxel'

const Jump = Input.createAction({ key: ' ' })

function PlayerBody() {
  const body = useRigidBody()
  const jump = useAction(Jump)

  useUpdate(() => {
    if (jump.justPressed()) body.applyImpulse([0, -320])
  })

  return (
    <body ref={body}>
      <collider shape={shapes.rectangle(24, 32)} group={['player']} collidesWith={['ground']} />
    </body>
  )
}
```

## `useCamera()`

```ts
useCamera(): CameraReference
```

`CameraReference` extends `Node2DReference<PrimaryNode.Camera>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `zoom` | `SignalGetter<Vector2>` | Current zoom. |
| `offset` | `SignalGetter<Vector2>` | Current camera offset. |
| `smoothing` | `SignalGetter<number>` | Current smoothing value. |
| `limit` | `SignalGetter<Bounds \| null>` | Current camera bounds. |
| `makeCurrent()` | `() => void` | Makes this the active camera. |
| `shake(options)` | `({ duration, strength }) => void` | Starts camera shake. |
| `screenToWorld(pos)` | `(pos: VectorLike) => Vector2` | Converts screen coordinates to world coordinates. |
| `worldToScreen(pos)` | `(pos: VectorLike) => Vector2` | Converts world coordinates to screen coordinates. |

```tsx
import { useCamera, useTrigger } from 'fraxel'

function MainCamera() {
  const camera = useCamera()

  useTrigger(camera.started, () => {
    camera.makeCurrent()
  })

  return <camera ref={camera} smoothing={8} />
}
```

## `useClickable()`

```ts
useClickable(): ClickableReference
```

`ClickableReference` extends `Node2DReference<PrimaryNode.Clickable>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `hovered` | `SignalGetter<boolean>` | `true` when pointer is inside. |
| `disabled` | `SignalGetter<boolean>` | Current disabled state. |
| `mousePosition` | `SignalGetter<Vector2>` | Pointer position in local coordinates. |
| `clicked` | `Trigger<[position: Vector2]>` | Fires on pointer release inside the area. |

```tsx
  useTrigger(clickable.clicked, (position) => {
    console.log('clicked at', position)
  })

  return <clickable ref={clickable} size={[120, 40]} />
}
```

## `useRayCast()`

```ts
useRayCast(): RayCastReference
```

`RayCastReference` extends `Node2DReference<PrimaryNode.RayCast>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `direction` | `SignalGetter<Vector2>` | Current ray direction. |
| `collider` | `SignalGetter<Collider \| null>` | Current detected collider. |
| `detected` | `SignalGetter<boolean>` | `true` when a collider is detected. |
| `colliderEntered` | `Trigger<[collider: Collider]>` | Fires when detection starts. |
| `colliderExited` | `Trigger<[collider: Collider]>` | Fires when detection ends. |
| `getCollider()` | `() => Collider \| null` | Returns the current detected collider. |

```tsx
import { useRayCast, useEffect } from 'fraxel'

function VisionRay() {
  const ray = useRayCast()

  useEffect(() => {
    if (ray.detected()) console.log('Detected:', ray.collider())
  })

  return <raycast ref={ray} direction={[200, 0]} collidesWith={['enemy']} />
}
```

## `useAnimation()`

```ts
useAnimation(): AnimationReference
```

`AnimationReference` extends `NodeReference<PrimaryNode.AnimationPlayer>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `animName` | `SignalGetter<string \| null>` | Current animation name. |
| `frameIndex` | `SignalGetter<number>` | Current frame index. |
| `ended` | `SignalGetter<boolean>` | `true` after the current animation ends. |
| `animationChanged` | `Trigger<[newAnim: string, oldAnim: string \| null]>` | Fires when animation changes. |
| `animationStopped` | `Trigger<[anim: string]>` | Fires when animation stops. |
| `animationIndexChanged` | `Trigger<[index: number]>` | Fires when frame index changes. |
| `animationEnded` | `Trigger<[anim: string]>` | Fires when animation ends. |
| `play(name, index?)` | `(name: string, index?: number) => void` | Plays an animation. |
| `stop()` | `() => void` | Stops the current animation. |
| `setNext(name)` | `(name: string \| null) => void` | Queues the next animation. |

```tsx
import { useAnimation, useTrigger } from 'fraxel'

function Animator() {
  const anim = useAnimation()

  useTrigger(anim.animationEnded, () => {
    anim.play('idle')
  })

  return <animator ref={anim} animations={() => animations} currentAnim="idle" />
}
```

## `useAudio()`

```ts
useAudio(): AudioReference
```

`AudioReference` extends `NodeReference<PrimaryNode.AudioPlayer>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `playing` | `SignalGetter<boolean>` | `true` while audio is playing. |
| `ended` | `Trigger<[]>` | Fires when playback finishes naturally. |
| `error` | `Trigger<[err: Error]>` | Fires when playback fails. |
| `play(offset?)` | `(offset?: number) => void` | Starts playback. |
| `pause()` | `() => void` | Pauses playback. |
| `stop()` | `() => void` | Stops playback and resets offset. |

```tsx
import SHOOT from '../assets/shoot.ogg?sound'
import { useAudio, useTrigger } from 'fraxel'

function SoundEffect() {
  const audio = useAudio()

  useTrigger(audio.ended, () => console.log('done'))

  return <audio ref={audio} soundId={SHOOT} />
}
```

## `useTimer()`

```ts
useTimer(): TimerReference
```

`TimerReference` extends `NodeReference<PrimaryNode.Timer>`.

| Member | Type | Description |
| ------ | ---- | ----------- |
| `time` | `SignalGetter<number>` | Current elapsed time in seconds. |
| `duration` | `SignalGetter<number>` | Current duration in seconds. |
| `progress` | `SignalGetter<number>` | Current `time / duration`, usually `0` to `1`. |
| `timeout` | `Trigger<[]>` | Fires when timer reaches duration. |
| `timeChanged` | `Trigger<[time: number]>` | Fires while timer is playing. |
| `play(from?)` | `(from?: number) => void` | Starts or resumes the timer. |
| `pause()` | `() => void` | Pauses without reset. |
| `stop()` | `() => void` | Stops and resets to `0`. |

```tsx
import { useTimer, useTrigger } from 'fraxel'

function Cooldown() {
  const timer = useTimer()

  useTrigger(timer.timeout, () => console.log('ready'))

  return <timer ref={timer} duration={3} autoPlay />
}
```

## `useView()`

```ts
useView(): ViewReference
```

`ViewReference` extends `NodeReference<PrimaryNode.View>` and adds no extra members. Use it for lifecycle access to `<view>` nodes.

```tsx
import { useView } from 'fraxel'

function HUD() {
  const view = useView()

  return (
    <view ref={view}>
      <text text="Score: 0" />
    </view>
  )
}
```
