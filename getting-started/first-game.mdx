The best way to understand how Fraxel works is to build something real. This guide walks you through creating a playable platformer — a character that runs, jumps, and lands on platforms with real physics. By the end you will have used the core building blocks: JSX scenes, rigid-body physics, input actions, and camera follow.

## Prerequisites

Make sure you have [installed Fraxel and configured TypeScript](/getting-started/installation) before continuing.

## Step 1 — Bootstrap the game

Create `src/main.tsx` with the minimal game setup. `createGame` takes a `<Game>` JSX element and a DOM container, wires up the canvas, and returns game controls.

```tsx src/main.tsx
import { createGame, Game, Scene } from 'fraxel/jsx'

function Main() {
  return <transform />
}

const game = createGame(
  <Game width={480} height={320} defaultScene="main">
    <Scene name="main" component={Main} />
  </Game>,
  document.querySelector('#root')!,
)

game.play()
```

- **`<Game width height defaultScene>`** — declares the canvas resolution and which scene to load first.
- **`<Scene name component>`** — registers a named scene backed by a component function.
- **`createGame(jsx, root)`** — processes the JSX, creates the canvas inside `#root`, and returns game controls.
- **`game.play()`** — starts the render loop.

## Step 2 — Add platforms

Static platforms are rigid bodies with `isStatic`. They don't move but still participate in collisions — other bodies bounce off them.

```tsx src/main.tsx
import { createGame, Game, Scene } from 'fraxel/jsx'
import { shapes } from 'fraxel'

function Platform({ position, width }: { position: [number, number]; width: number }) {
  return (
    <rigid-body isStatic position={position}>
      <geometry shape={shapes.rectangle(width, 16)} fillColor={[0.2, 0.3, 1, 1]} />
      <collider shape={shapes.rectangle(width, 16)} group={['ground']} collidesWith={['player']} />
    </rigid-body>
  )
}

function Main() {
  return (
    <transform>
      <Platform position={[0, 300]} width={480} />
      <Platform position={[80, 220]} width={100} />
      <Platform position={[280, 160]} width={120} />
      <Platform position={[140, 100]} width={100} />
    </transform>
  )
}

const game = createGame(
  <Game width={480} height={320} defaultScene="main">
    <Scene name="main" component={Main} />
  </Game>,
  document.querySelector('#root')!,
)

game.play()
```

`<geometry>` renders a colored shape without needing textures. The `fillColor` is `[r, g, b, a]` with values from `0` to `1`.

## Step 3 — Add a player with physics

A `<rigid-body>` node adds gravity, velocity, and collision response. It **must** contain a `<collider>` child that defines the physical shape.

```tsx src/main.tsx
import { createGame, Game, Scene } from 'fraxel/jsx'
import { shapes } from 'fraxel'
import { useRigidBody } from 'fraxel/hooks'

function Platform({ position, width }: { position: [number, number]; width: number }) {
  return (
    <rigid-body isStatic position={position}>
      <geometry shape={shapes.rectangle(width, 16)} fillColor={[0.2, 0.3, 1, 1]} />
      <collider shape={shapes.rectangle(width, 16)} group={['ground']} collidesWith={['player']} />
    </rigid-body>
  )
}

function Player() {
  const body = useRigidBody()

  return (
    <rigid-body ref={body} position={[220, 0]} mass={2}>
      <geometry shape={shapes.capsule(12, 8)} fillColor={[1, 0.1, 0.4, 1]} />
      <collider shape={shapes.capsule(12, 8)} group={['player']} collidesWith={['ground']} />
    </rigid-body>
  )
}

function Main() {
  return (
    <transform>
      <Platform position={[0, 300]} width={480} />
      <Platform position={[80, 220]} width={100} />
      <Platform position={[280, 160]} width={120} />
      <Platform position={[140, 100]} width={100} />
      <Player />
    </transform>
  )
}

const game = createGame(
  <Game width={480} height={320} defaultScene="main">
    <Scene name="main" component={Main} />
  </Game>,
  document.querySelector('#root')!,
)

game.play()
```

The player spawns at `[220, 0]` and immediately falls under gravity. When it hits a platform, the physics system resolves the collision — no manual collision detection needed.

## Step 4 — Add input handling

Fraxel's input system maps keys to semantic actions. `useAction` gives you reactive pressed/justPressed state, and `useActionAxis` returns `-1`, `0`, or `1` based on two opposing keys.

```tsx src/main.tsx
import { createGame, Game, Scene } from 'fraxel/jsx'
import { Input, shapes } from 'fraxel'
import { useRigidBody, useAction, useActionAxis, useEffect, useUpdate } from 'fraxel/hooks'

const Left = Input.createAction({ key: 'a' })
const Right = Input.createAction({ key: 'd' })
const Jump = Input.createAction({ key: ' ' })

function Platform({ position, width }: { position: [number, number]; width: number }) {
  return (
    <rigid-body isStatic position={position}>
      <geometry shape={shapes.rectangle(width, 16)} fillColor={[0.2, 0.3, 1, 1]} />
      <collider shape={shapes.rectangle(width, 16)} group={['ground']} collidesWith={['player']} />
    </rigid-body>
  )
}

function Player() {
  const body = useRigidBody()
  const direction = useActionAxis(Left, Right)
  const jump = useAction(Jump)

  useEffect(() => {
    if (jump.justPressed() && body.isGrounded()) {
      body.applyImpulse([0, -400])
    }
  })

  useUpdate(() => {
    body.setVelocity([direction() * 240, body.velocity().y])
  })

  return (
    <rigid-body ref={body} position={[220, 0]} mass={2}>
      <geometry shape={shapes.capsule(12, 8)} fillColor={[1, 0.1, 0.4, 1]} />
      <collider shape={shapes.capsule(12, 8)} group={['player']} collidesWith={['ground']} />
    </rigid-body>
  )
}

function Main() {
  return (
    <transform>
      <Platform position={[0, 300]} width={480} />
      <Platform position={[80, 220]} width={100} />
      <Platform position={[280, 160]} width={120} />
      <Platform position={[140, 100]} width={100} />
      <Player />
    </transform>
  )
}

const game = createGame(
  <Game width={480} height={320} defaultScene="main">
    <Scene name="main" component={Main} />
  </Game>,
  document.querySelector('#root')!,
)

game.play()
```

Here's what's happening:

- **`Input.createAction({ key: 'a' })`** — creates a reusable action symbol bound to a key.
- **`useActionAxis(Left, Right)`** — returns `-1` when moving left, `1` when right, `0` when neither.
- **`useAction(Jump)`** — gives you `justPressed()` which is `true` only on the first frame the key is held.
- **`body.isGrounded()`** — reactive `true` when the body rests on a surface.
- **`body.applyImpulse([0, -400])`** — instant upward velocity for jumping.
- **`body.setVelocity([x, y])`** — overrides velocity each frame for horizontal movement.

## Step 5 — Add camera follow

The `<camera>` node follows a target and applies viewport transforms. Attach it to the player so the view tracks their movement.

```tsx src/main.tsx
import { createGame, Game, Scene } from 'fraxel/jsx'
import { Input, shapes } from 'fraxel'
import { useRigidBody, useAction, useActionAxis, useEffect, useUpdate } from 'fraxel/hooks'

const Left = Input.createAction({ key: 'a' })
const Right = Input.createAction({ key: 'd' })
const Jump = Input.createAction({ key: ' ' })

function Platform({ position, width }: { position: [number, number]; width: number }) {
  return (
    <rigid-body isStatic position={position}>
      <geometry shape={shapes.rectangle(width, 16)} fillColor={[0.2, 0.3, 1, 1]} />
      <collider shape={shapes.rectangle(width, 16)} group={['ground']} collidesWith={['player']} />
    </rigid-body>
  )
}

function Player() {
  const body = useRigidBody()
  const direction = useActionAxis(Left, Right)
  const jump = useAction(Jump)

  useEffect(() => {
    if (jump.justPressed() && body.isGrounded()) {
      body.applyImpulse([0, -400])
    }
  })

  useUpdate(() => {
    body.setVelocity([direction() * 240, body.velocity().y])
  })

  return (
    <rigid-body ref={body} position={[220, 0]} mass={2}>
      <camera current smoothing={5} offset={[0, -30]} />
      <geometry shape={shapes.capsule(12, 8)} fillColor={[1, 0.1, 0.4, 1]} />
      <collider shape={shapes.capsule(12, 8)} group={['player']} collidesWith={['ground']} />
    </rigid-body>
  )
}

function Main() {
  return (
    <transform>
      <Platform position={[0, 300]} width={480} />
      <Platform position={[80, 220]} width={100} />
      <Platform position={[280, 160]} width={120} />
      <Platform position={[140, 100]} width={100} />
      <Player />
    </transform>
  )
}

const game = createGame(
  <Game width={480} height={320} defaultScene="main">
    <Scene name="main" component={Main} />
  </Game>,
  document.querySelector('#root')!,
)

game.play()
```

- **`<camera current>`** — makes this the active camera that controls the viewport.
- **`smoothing={5}`** — camera interpolation speed (higher = snappier follow).
- **`offset={[0, -30]}`** — camera offset so the player isn't centered at the bottom.

## What you learned

In under 80 lines you built a platformer with:

- **Physics** — gravity, collision response, and jumping via `applyImpulse`
- **Input** — semantic actions with `Input.createAction` and reactive state with `useAction`
- **Platforms** — static rigid bodies that the player lands on
- **Camera** — smooth follow attached to the player
- **Geometry** — colored shapes rendered without textures
