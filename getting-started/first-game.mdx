This tutorial starts from the `Empty` template and turns it into a small collectible game. You will edit a scene, create reusable components, add input, use physics, and react to collisions.

You do not need to call `Game.setup()` or create engine objects manually. The template already gives you Vite, `@fraxel/vite-plugin`, JSX, and `createGame()` wiring.

## 1. Create an Empty project

Create a new project and choose the `Empty` template when prompted:

```bash
npx create-fraxel my-first-game
```

Then run it:

```bash
cd my-first-game
npm install
npm run dev
```

Open `src/scenes/main.tsx`. This is where you will build the game scene.

## 2. Draw a player and floor

Replace the scene with a player body and a static floor:

```tsx src/scenes/main.tsx
import { shapes } from 'fraxel'

function Player() {
  return (
    <body position={[80, 40]} mass={1}>
      <geometry shape={shapes.rectangle(24, 32)} fillColor={[0.2, 0.8, 1, 1]} />
      <collider
        shape={shapes.rectangle(24, 32)}
        group={['player']}
        collidesWith={['ground', 'coin']}
      />
    </body>
  )
}

function Floor() {
  return (
    <body position={[0, 220]} isStatic>
      <geometry shape={shapes.rectangle(480, 24)} fillColor={[0.25, 0.25, 0.3, 1]} />
      <collider shape={shapes.rectangle(480, 24)} group={['ground']} collidesWith={['player']} />
    </body>
  )
}

export default function MainScene() {
  return (
    <transform>
      <Floor />
      <Player />
    </transform>
  )
}
```

What you used:

- `<body>` adds physics behavior.
- `<collider>` defines what participates in collision.
- `<geometry>` draws simple shapes without needing art assets yet.
- `shapes.rectangle()` creates matching render and collision shapes.

## 3. Add movement with hooks

Fraxel projects usually interact with nodes through hooks. `useRigidBody()` gives you a typed reference to the `<body>` node, and `useActionAxis()` converts two actions into a `-1`, `0`, or `1` movement value.

```tsx src/scenes/main.tsx
import { Input, shapes, useAction, useActionAxis, useRigidBody, useUpdate } from 'fraxel'

const MoveLeft = Input.createAction({ key: 'a' })
const MoveRight = Input.createAction({ key: 'd' })
const Jump = Input.createAction({ key: ' ' })

function Player() {
  const body = useRigidBody()
  const x = useActionAxis(MoveLeft, MoveRight)
  const jump = useAction(Jump)

  useUpdate(() => {
    const velocity = body.velocity()
    body.setVelocity([x() * 160, velocity.y])

    if (jump.justPressed()) {
      body.applyImpulse([0, -320])
    }
  })

  return (
    <body ref={body} position={[80, 40]} mass={1}>
      <geometry shape={shapes.rectangle(24, 32)} fillColor={[0.2, 0.8, 1, 1]} />
      <collider
        shape={shapes.rectangle(24, 32)}
        group={['player']}
        collidesWith={['ground', 'coin']}
      />
    </body>
  )
}
```

The player now moves with `A` and `D`, and jumps with Space.

## 4. Add a collectible

Use `useCollider()` and `useTrigger()` to react when the player touches a coin.

```tsx src/scenes/main.tsx
import {
  Input,
  shapes,
  useAction,
  useActionAxis,
  useCollider,
  useRigidBody,
  useSignal,
  useTrigger,
  useUpdate,
} from 'fraxel'

const MoveLeft = Input.createAction({ key: 'a' })
const MoveRight = Input.createAction({ key: 'd' })
const Jump = Input.createAction({ key: ' ' })

function Coin({ onCollect }: { onCollect: () => void }) {
  const collider = useCollider()

  useTrigger(collider.colliderEntered, () => {
    onCollect()
    collider.node.destroy()
  })

  return (
    <collider
      ref={collider}
      position={[260, 180]}
      shape={shapes.circle(10)}
      group={['coin']}
      collidesWith={['player']}>
      <geometry shape={shapes.circle(10)} fillColor={[1, 0.85, 0.2, 1]} />
    </collider>
  )
}

function Player() {
  const body = useRigidBody()
  const x = useActionAxis(MoveLeft, MoveRight)
  const jump = useAction(Jump)

  useUpdate(() => {
    const velocity = body.velocity()
    body.setVelocity([x() * 160, velocity.y])

    if (jump.justPressed()) {
      body.applyImpulse([0, -320])
    }
  })

  return (
    <body ref={body} position={[80, 40]} mass={1}>
      <geometry shape={shapes.rectangle(24, 32)} fillColor={[0.2, 0.8, 1, 1]} />
      <collider
        shape={shapes.rectangle(24, 32)}
        group={['player']}
        collidesWith={['ground', 'coin']}
      />
    </body>
  )
}

function Floor() {
  return (
    <body position={[0, 220]} isStatic>
      <geometry shape={shapes.rectangle(480, 24)} fillColor={[0.25, 0.25, 0.3, 1]} />
      <collider shape={shapes.rectangle(480, 24)} group={['ground']} collidesWith={['player']} />
    </body>
  )
}

export default function MainScene() {
  const [score, setScore] = useSignal(0)

  return (
    <transform>
      <text position={[16, 16]} text={() => `Score: ${score()}`} />
      <Floor />
      <Coin onCollect={() => setScore(score() + 1)} />
      <Player />
    </transform>
  )
}
```

What you added:

- `useSignal()` stores score.
- `useCollider()` gives access to collision triggers.
- `useTrigger()` subscribes to `colliderEntered` and cleans up automatically.
- `collider.node.destroy()` removes the collected coin.

## 5. Replace shapes with sprites

When you add art, place assets under `src/assets/` and import them with `?texture`:

```tsx
import PLAYER from '../assets/player.png?texture'

function PlayerSprite() {
  return <sprite textureId={PLAYER} displaySize={[24, 32]} />
}
```

Then replace the player's `<geometry>` with `<PlayerSprite />`.

The Vite plugin turns `?texture` imports into Fraxel texture symbols. Use the same pattern for sounds with `?sound`.
